1. Установка проекта производится при помощи импорта исходных файлов в Eclipse из репозитория GitHub.
	https://github.com/viktorkorogoda/inteticsTestProject
	
2. База данных создаётся из файла дампа базы данных, который находится в папке с проектом.

3. В файле serviceStation.properties указываются параметры для подключения к БД.
